# paradigmachallenge

## Uitvoeren van de code
Om deze code uit te voeren moet je elixir installeren: https://elixir-lang.org/install.html.

Nadat elixir is geinstalleerd kunnen de bestanden met de volgende commandos worden uitgevoerd:
`elixir rle_compress_runner.exs`
`elixir rle_decompress_runner.exs`
`elixir rle_test.exs`
